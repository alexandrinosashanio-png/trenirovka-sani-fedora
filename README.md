# Тренировочка для Сани и Фёдора

Трёхнедельная программа тренировок для двух юных футболистов (14 лет): гантели, штанга, скамья.

**Открыть страницу:** https://alexandrinosashanio-png.github.io/trenirovka-sani-fedora/

## Что внутри
- 3 тренировочных дня (ноги+кор, верх тела, всё тело+сила) с описанием техники каждого упражнения
- Разминка перед каждой тренировкой
- Таймер (планка, отдых) со звуком и вибрацией
- Ссылки на русскоязычные видео по технике
- Прогрессия по неделям и идеи для соревнования на двоих
- Кнопка «Скачать страницу» для офлайн-доступа и инструкция «На экран Домой» для iPhone

Одностраничный сайт, весь код в `index.html`, без зависимостей.
