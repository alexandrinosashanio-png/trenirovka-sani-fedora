# PROJECT_STATE — Тренировочка для Сани и Фёдора

## Purpose
Одностраничный сайт с трёхнедельной программой тренировок для двух юных футболистов (~14 лет): гантели, штанга, скамья. Цель — дать понятную программу с акцентом на технику и возможность держать её под рукой в зале (в т.ч. офлайн и ярлыком на телефоне).

## Environment & commands
- OS: Windows 11, оболочка PowerShell
- Стек: чистый HTML/CSS/JS, без зависимостей и сборки. Весь сайт в `index.html`
- Открыть локально: открыть `index.html` в браузере
- Опубликовано: GitHub Pages (ветка `main`, корень)
- Обновить сайт: править `index.html` → `git add -A` → `git commit` → `git push` (Pages пересобирается автоматически)

## Architecture (current state)
- Единственный файл `index.html`: полноценный HTML-документ (doctype/head/body) + инлайн `<style>` + инлайн `<script>`.
- В `<head>` — meta для iOS (`apple-mobile-web-app-*`, `viewport-fit=cover`, `theme-color`), favicon и apple-touch-icon как SVG data-URI (⚽). Благодаря этому ярлык «На экран Домой» на iPhone открывается на весь экран.
- Шрифты — системные (Impact/Arial Narrow как аналоги Anton/Barlow Condensed), внешних CDN нет: работает офлайн.
- Дизайн: тёмная «газонная» тема, осознанно single-theme.
- Секции: Hero → «Читать перед стартом» → Разминка → Таймер → Дни A/B/C с описанием техники → «Техника на видео» (ссылки на YouTube-поиск на русском) → «Сохранить на телефон» → «Чтобы было интересно» → «Прогресс по неделям» → футер.
- Таймер (`<script>`): обратный отсчёт с SVG прогресс-кольцом, пресеты 30/40/60/90 сек, звук через Web Audio API (без файлов), вибрация через `navigator.vibrate`.
- Кнопка «Скачать страницу»: `document.documentElement.outerHTML` → Blob → download, даёт рабочий офлайн-файл.

## Git-решение
- РЕШЕНИЕ (2026-07-07): проект размещён на GitHub, репозиторий публичный. Причина: нужна стабильная публичная ссылка для передачи третьим лицам без входа/расшаривания; плюс история и лёгкие обновления. Секретов в дереве нет. Больше не переспрашивать про заливку — она уже настроена.
